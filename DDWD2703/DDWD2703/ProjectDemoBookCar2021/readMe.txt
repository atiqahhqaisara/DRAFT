HOW TO RUN CAR RENTAL SYSTEM DEMO
==================================
1. You will need to create a user for in PHPMyAdmin:-

username : web2
password : web2

2. Create a database in phpmyadmin named cardb


3. in the sql tab in PHPMyAdmin, import the tables and data using the cardb.sql file
	- this will create 4 table:-
		Table - car, customers, bookings , users

3. to run, open the page ProjectDemoBookCar2021/mainMenu.php on your browser
- the login form is embedded in the mainMenu.php, for your project you can follow the same format or have a separate login page
